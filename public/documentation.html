MyFlix API v1.0.0

The MyFlix API is built with the REST architecture in mind. This API has self-explanatory URLs, returns JSON responses, and uses standard HTTP response codes.


Supported Requests

Request URL HTTP Method Request body data format Response body data format
Get list of movies /movies GET None An array of JSON objects holding data about all the movies in the database
Get list of movies by director /movies/directors/[director name] GET None An array of JSON objects holding data about all the movies by requested director
Get list of movies by genre /movies/genres/[genre] GET None An array of JSON objects holding data about all the movies by requested genre
Get data about a single movie by movieName /movies/[movieName] GET None A JSON object holding data about the requested movie like this:
{ "genre": "Sci-Fi",
"director": "Steven Spielberg",
"movieid": "4",
"birthdate": "1946-12-18T00:00:00.000Z",
"description": "An epic science fiction adventure that transports you to new worlds.",
"movieName": "Echoes of Eternity",
"bio": "Steven Spielberg is an American film director, producer, and screenwriter. He is considered one of the founding pioneers of the New Hollywood era."
}
Add a user /users POST A JSON object holding data about the user to add, structured like:

{ _id: ObjectId('65cdc96e07f819554ce29be4'),
firstName: 'Mittal',
lastName: 'Malankiya',
email: 'm4mittal13@gmail.com',
password: '87hvhsc563',
favorite_movie: [ ObjectId('65c9fef2a71ad74d5d8083c7') ],
birthday: ISODate('1998-04-23T00:00:00.000Z'),
__v: 0 } 		A JSON object holding data about the user that was added, including their assigned id:
_id: ObjectId('65cdc96e07f819554ce29be4'),
firstName: 'Piyush',
lastName: 'Malankiya',
email: 'm4mittal13@gmail.com',
password: '87hvhsc563',
favorite_movie: [ ObjectId('65c9fef2a71ad74d5d8083c7') ],
birthday: ISODate('1998-04-23T00:00:00.000Z'),
__v: 0
Add a movie object to a user's favorites list /users/[username]/movies/[movieID] PUT None A text message indicating whether the movie was successfully added from the user's favorited movies
Remove a movie from a user's favorites list /users/[username]/movies/[movieID] DELETE None A text message indicating whether the movie was successfully removed from the user's favorited movies
